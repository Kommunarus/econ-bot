**Python‑скрипт**

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-

import csv
import matplotlib.pyplot as plt

# 1. Создаём CSV‑файл
csv_path = "../sandbox/organisms.csv"
data = [
    ["организм", "длина (мм)"],
    ["амеба",      0.1],
    ["инфузория",  0.3],
    ["эвглена",    0.05],
    ["трубач",     1.0],
    ["коловратка", 1.0]
]

with open(csv_path, "w", newline="", encoding="utf-8") as f:
    writer = csv.writer(f)
    writer.writerows(data)

# 2. Читаем CSV и строим график
organisms = []
lengths   = []

with open(csv_path, newline="", encoding="utf-8") as f:
    reader = csv.DictReader(f)
    for row in reader:
        organisms.append(row["организм"])
        lengths.append(float(row["длина (мм)"]))

plt.figure(figsize=(6, 4))
bars = plt.bar(organisms, lengths, color="#4c72b0")
plt.ylabel("Длина (мм)")
plt.title("Размеры микроорганизмов")

# подписи над столбцами
for bar in bars:
    height = bar.get_height()
    plt.text(bar.get_x() + bar.get_width()/2.0, height,
             f'{height:.2f}', ha='center', va='bottom')

# 3. Сохраняем график
output_path = "../sandbox/g.png"
plt.tight_layout()
plt.savefig(output_path)
print(f"График сохранён в {output_path}")
```

**Что делает скрипт**

1. Создаёт CSV‑файл `../sandbox/organisms.csv` с двумя колонками: название и длина (мм).
2. Читает этот файл, собирает данные в списки.
3. Строит столбчатую диаграмму, добавляя подписи над каждым столбцом.
4. Сохраняет изображение как `../sandbox/g.png`.

Запустите скрипт в песочнице (Python 3 + matplotlib). После выполнения вы найдёте файл `g.png` с графиком размеров микроорганизмов.
