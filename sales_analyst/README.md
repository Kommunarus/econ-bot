# Sales Analyst

Этот бот предназначен для анализа данных о продажах с использованием Python. 

## Структура проекта

- **main.py** — основной скрипт для запуска анализа данных.
- **requirements.txt** — список необходимых библиотек Python.
- **.env** — файл конфигурации переменных окружения (например, пути к данным).
- **logi.log** — логи выполнения программы.
- **sandbox/** — директория с результатами и временными файлами:
   - `sales_2024.csv` — пример входного файла с данными о продажах за 2024 год.

## Установка и запуск

1. Убедитесь, что у вас установлен Python 3.12+ и активировано виртуальное окружение.
2. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

3. Запустите анализ:

   ```bash
   python main.py
   ```

## Функциональность

Скрипт выполняет следующие задачи:
- Запуск агента которому доступны 2 инструмента - выполнение python скрипта и чтение файлов с диска. 
- Можно задавать вопросы по файлу `sales_2024.csv` и агент будет на них отвечать.
- Логирование процесса выполнения в `logi.log`.


## Примеры данных

В папке `sandbox/` находится файл `sales_2024.csv`, который можно использовать как шаблон структуры входных данных

## Переменные окружения

Для работы приложения необходимо настроить `.env` файл с параметрами.
